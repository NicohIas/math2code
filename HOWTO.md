#

## The config:

```
{
    "encryptSetting": "True",
    "bypassKeySetting": "False",
    "bypassTagSetting": "False",
    "skipSetting": "False"
}
```

Example:

```
{
    "encryptSetting": "False", <<
    "bypassKeySetting": "False",
    "bypassTagSetting": "False",
    "skipSetting": "False"
}
```
